# Session

[Live demo](https://session.alkimcaner.com/)

Built with:

- WebRTC
- PeerJS
- React
- Redux
- React Router
- Typescript
- Supabase
- Tailwindcss

## Project Setup

### Install dependencies

```bash
pnpm install
```

### Start dev server

```bash
pnpm dev
```

### Build for production

```bash
pnpm build
```
