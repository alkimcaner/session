# Session

[Live demo](https://session.alkimcaner.com/)

Built with:

- WebRTC
- Next.js
- React
- Redux
- Typescript
- PostgreSQL
- Supabase
- Tailwindcss

## Screenshot

![gif](https://github.com/alkimcaner/portfolio/blob/main/public/assets/session.jpg)

## Project Setup

### Install dependencies

```bash
npm install
# or
yarn install
```

### Start dev server

```bash
npm run dev
# or
yarn dev
```

### Build for production

```bash
npm run build
# or
yarn build
```
